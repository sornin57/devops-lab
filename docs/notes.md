# Notes DevOps

## Docker

Docker permet de lancer une application dans un container avec ses dépendances.

- Image : modèle prêt à lancer
- Container : instance lancée d'une image
- Dockerfile : recette pour construire une image
- Docker Hub : registre public où Docker peut télécharger des images

## Commandes vues

- docker --version
- docker run hello-world
- docker build
- docker run
- docker ps
- docker logs
